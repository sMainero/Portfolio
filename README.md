# Portfolio

Interactive 3D portfolio inspired by old Pokémon games and the Game Boy Color.

Built with vanilla JavaScript and Three.js, inside a fully rendered 3D Game Boy environment.

## About

This started as a fun experiment combining:

* retro game aesthetics
* 3D graphics
* frontend interaction
* shaders and rendering
* portfolio design

Instead of a traditional portfolio site, I wanted something that felt more like exploring a game world.

## Features

* fully interactive 3D Game Boy Color
* Pokémon inspired UI and movement
* keyboard controls
* realtime rendering with Three.js
* retro pixel art style
* custom shaders and effects
* responsive fullscreen experience

## Stack

* Vanilla JavaScript
* Three.js
* HTML/CSS
